cách chạy forumweb
```bash
cd forumweb
```

```bash
pip install virtualenv
```

```bash
virtualenv env
```

```bash
env\scripts\activate
```

```bash
pip install -r requirements.txt
```

run server : 
```bash
python manage.py runserver
```
